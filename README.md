# BusLog App

A small app that lets you keep a record of the transportation you've travelled.

## Functions

- Material 3 UI.
- `SQLite` based, no internet access required.
- Auto-category based on dates.
- Auto-complete available when adding a record.

### Languages

Currently support:

- English (default)
- Simplified Chinese
- Traditional Chinese
